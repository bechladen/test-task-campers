# TravelTrucks — фронтенд застосунок (React)

Мета проєкту — створити фронтенд частину веб‑додатку для компанії **TravelTrucks**, яка займається орендою кемперів.

## Функціонал

- **Домашня сторінка** (`/`): hero‑банер + CTA **View Now** → перехід у каталог.
- **Каталог** (`/catalog`):
  - список кемперів
  - фільтрація за **локацією**, **типом кузова**, **equipment** (AC/кухня/TV/ванна тощо)
  - **обране** (favorites) збережене між перезавантаженнями сторінки
  - **Load more** — довантаження карток з урахуванням фільтрів
  - **Show more** відкриває сторінку деталей у **новій вкладці**
- **Сторінка кемпера** (`/catalog/:id`):
  - галерея фото
  - вкладки **Features / Reviews**
  - форма бронювання з валідацією та нотифікацією про успіх

## API

Використовується готовий бекенд API:

- `GET /campers`
- `GET /campers/:id`

Base URL: `https://66b1f8e71ca8ad33d4f5f63e.mockapi.io`

## Технології

- **React** + **Vite**
- **React Router**
- **Redux Toolkit** + **redux-persist**
- **Axios**
- **Formik** + **Yup** (форма бронювання)
- **react-hot-toast** (нотифікації)

## Запуск проєкту локально

1. Встановити залежності:

```bash
npm install
```

2. Запустити dev‑сервер:

```bash
npm run dev
```

3. Збірка:

```bash
npm run build
```

4. Перевірка лінтера / форматування:

```bash
npm run lint
npm run format-check
```
